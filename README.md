# Calculadora App
Um aplicativo de calculadora mobile desenvolvido com **React Native**. Este projeto foi criado para ser uma ferramenta utilitária leve, focada em uma experiência de usuário simples e eficiente, demonstrando o poder do desenvolvimento cross-platform.
## Funcionalidades

- Operações aritméticas fundamentais (soma, subtração, multiplicação e divisão).
- Interface intuitiva com botões de fácil interação.
- Gerenciamento de estado em tempo real para exibição de números.
- Layout responsivo que se ajusta a diferentes densidades de tela.

## Tecnologias Utilizadas
| Tecnologia | Descrição |
| --- | --- |
| React Native | Framework para desenvolvimento mobile nativo |
| Expo | Plataforma de auxílio ao desenvolvimento e build |
| JavaScript | Lógica de programação do aplicativo |
| StyleSheet | Estilização de componentes (CSS-in-JS) |
> [!TIP]
> Por ser um projeto desenvolvido com Expo, você pode testar o app diretamente no seu celular instalando o aplicativo "Expo Go" e escaneando o QR Code gerado ao iniciar o projeto.
## Execução Local
Certifique-se de ter o Node.js instalado. No terminal, execute:
`npm install`
Para iniciar o servidor de desenvolvimento:
```bash

npx expo start

```
